感觉的类型
...

  • 特殊感觉(special sense)：视、听、嗅、味、平衡觉
  • 躯体感觉(somatosensation)：触、压、冷、热、痛觉
    • 精细触觉：两点辨别和物质纹理的粗细
  • 本体感觉(proprioception)/深感觉：位置觉、运动觉、震动觉
    • 意识性：肌肉张力和关节位置的感觉
    • 非意识性：脊髓对小脑的传入，调节肌张力和随意运动中发挥作用
  • 内脏感觉(visceral sense)：主要是痛觉，还有一些机械感觉 (如胃肠收缩、膀胱充盈、血压的升降等)
  • 化学感觉：如血液酸度、血浆渗透压、血浆葡萄糖浓度等

脊髓丘脑束
...

  • 传导痛、温、粗触觉、压觉
  • 第1级：背根神经节（DRG）
    • 感觉传入纤维
    • 双极神经元
  • 第2级：脊髓背角
    • 换元
    • 经中央管前交叉至对侧
    • 上行形成脊髓丘脑束
  • 第3级：丘脑腹后外侧核
    • 丘脑辐射，经内囊后肢
    • 中央后回中上部

内侧丘系
...

  • 传导精细触觉和（意识性）本体觉
  • 第1级：背根神经节（DRG）
    • 感觉传入纤维
    • 双极神经元
  • 第1.5级：脊髓背角
    • 换元/不换元
    • 同侧背柱(dorsal column)
    • 形成薄束(gracile tract)或楔束(cuneate tract)
  • 第2级：薄束核或楔束核
    • 换元
    • 纤维交叉到对侧
    • 形成内侧丘系
  • 第3级：丘脑腹后外侧核
    • 丘脑辐射，经内囊后肢
    • 中央后回中上部

三叉丘系
...

  • 头面部感觉
  • 第1级：三叉神经节
  • 第2级：
    • 中脑核：本体觉
    • 脑桥核：触压觉
    • 脊束核：痛温觉
    • 二级纤维跨越至对侧形成三叉丘系（部分不交叉）
  • 第3级：丘脑腹后内侧核
    • 丘脑中央辐射，经内囊后肢
    • 中央后回下部
