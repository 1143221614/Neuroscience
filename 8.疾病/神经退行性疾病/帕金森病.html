帕金森病
...

  • 帕金森病（Parkinson’s disease, PD）是常见中老年神经系统退行性疾病
  • 生化异常：纹状体多巴胺降低，多巴胺与乙酰胆碱递质失平衡
  • 病理表现：黑质多巴胺能神经元进行性退变和路易小体形成

左旋多巴
...

  • 多巴胺不能透过BBB，左旋多巴经过多巴脱羧酶转化为多巴胺

临床应用
...

  • 抗帕金森：对轻症患者更有效，随着病程延长疗效逐渐下降（不可逆的神经退行）
  • 治疗肝昏迷：转化为NE，促进意识清醒，但不能改善肝功

左旋多巴的中枢分布少，易在外周转化
...

  • 口服只有1%进入中枢发挥治疗作用
  • 在外周被降解：多巴胺脱羧酶（AADC），使得左旋多巴在外周转化为多巴胺

左旋多巴后期疗效不稳定
...

  • 表现：症状波动与开关现象
    • 症状波动
      • 服药后：患者运动功能正常（有疗效）
      • 药物代谢后：患者出现PD症状（无效）——剂末效应
    • “开－关现象”
      • 患者突然多动不安（开）
      • 随后又出现全身性或肌强直性不动状态（关）
  • 原因：疾病进展+左旋多巴血药浓度波动

不良反应
...

早期反应
...
  • 胃肠道反应：
    • 恶心、呕吐、食欲减退
    • 机制：
      • DA刺激胃粘膜
      • DA刺激延脑催吐化学感受区
    • 措施：饭后服药，多潘立酮（D2受体阻断剂）
  • 心血管反应：
    • 体位性低血压，心动过速或心律失常
    • 机制：DA舒张血管、兴奋心脏β1受体
    • 措施：β受体阻断剂
长期反应
...
  • 精神障碍：
    • 中枢兴奋——激动、失眠、焦虑、幻觉、幻视
    • 机制：兴奋中脑-边缘系统、中脑-皮质的DA通路
    • 措施：非典型抗精神病药物（氯氮平）
  • 不自主异常运动：
    • 咬牙、吐舌、点头、做怪相及舞蹈样动作
    • 机制：DA补充过度有关，间接通路的病理性激活
    • 措施：减量

左旋多巴增效剂
...

  • 与左旋多巴联合使用

增效机制
...

  • 抑制左旋多巴降解：COMT抑制剂-托卡朋、恩他卡朋
  • 抑制外周转化：外周AADC抑制剂-卡比多巴（不能透过BBB）
  • 抑制中枢降解：中枢MAO-B抑制剂-司来吉兰

卡多多巴
...

  • 外周脱羧酶抑制剂
  • 不能进中枢，对中枢的脱羧酶无效果
  • 单用无效，与左旋多巴合用，提高疗效、减少副作用

– 大体：中脑黑质蓝斑色素缺失

体成分、热休克蛋白等);常分布于黑质和蓝斑区，此外还可见于基底节、皮质、脑干、脊髓甚至
