α-运动神经元
...

  • α-运动神经元支配梭外肌，直接控制肌肉收缩，是骨骼肌运动反射的最后公路（final common path）
  • α-运动神经元位于脊髓前角：
    • 在脊髓节段分布
      • 上肢：颈膨大
      • 下肢：腰膨大
      • 近端和远端：颈段和腰骶段
      • 体轴肌肉：所有节段
    • 在脊髓前角内排列分布
      • 屈肌：前角的背侧
      • 伸肌：前角的腹侧
      • 体轴肌肉：前角的内侧
      • 远端肌肉：前角的外侧

运动单位与运动神经元池
...

  • 运动单位（motor unit）：一个α-运动神经元及其支配的所有肌纤维
    • 大小与功能相适应：
      • 运动单位小(如眼外肌)，有利于肌肉的精细运动
      • 运动单位大(如三角肌)，有利于产生巨大的肌肉收缩力
    • 交叉分布：
      • 同一个运动神经元的肌纤维，可与其他运动单位的肌纤维交叉分布
      • 即使只有少数运动神经元活动，肌肉产生的张力也是均匀的
  • 运动神经元池（motor neuron pool）：支配同一块肌肉的所有运动神经元

运动单位的类型
...

  • 慢速收缩型（slow, S型）
  • 快速收缩抗疲劳型（fast fatigue resistant , FFR型）
  • 快速收缩易疲劳型（fast fatigue, FF型）

α-运动神经元的放电频率与肌肉收缩
...

  • 动作电位发放频率较低时，产生非融合性强直收缩（力量在平台之下，没有达到最大值）
  • 动作电位发放频率增高后，产生融合性强直收缩（力量到达平台，即运动单位所能达到的最大力量）

α-运动神经元的输入信号
...

  • 脊髓中间神经元的输入：脊髓运动程序的神经环路
  • 肌梭的感觉输入：反馈关于肌肉长度的信息
  • 上运动神经元的输入：随意运动的发起和控制

腱器官
...

  • 肌肉和肌腱连接处的肌肉张力的本体感受器
  • 包裹于肌腱内的胶原纤维束内，与梭外肌成串联
  • 传入纤维：Ib类纤维
  • 效应：抑制脊髓前角支配该肌肉的α-运动神经元

脊髓反射
...

  • 脊髓对运动和姿势的调节是通过各种脊髓反射（spinal reflex）/姿势反射（postural reflex）

牵张反射
...

  • 牵张反射：有神经支配的骨骼肌，在受到外力牵张刺激时，能反射性地引起受牵拉的同一块肌肉收缩，是维持姿势及完成动的基础
腱反射肌紧张
牵张刺激快速牵拉肌腱缓慢持续牵拉肌腱
感受器肌梭（动态纤维）肌梭（静态纤维）
传入神经Ia类纤维II类纤维
反射弧单突触反射，潜伏期短多突触反射
收缩成分快肌纤维慢肌纤维
收缩特点同步性快速 力量大持续交替收缩，不易疲劳
生理意义快速收缩不表现为明显动作，维持姿势和肌张力
  • 牵张反射的病理反应
    • 减弱或消失：提示反射弧损害或中断
    • 亢进：高位中枢病变

腱器官反射
...

  • 反牵张反射/腱器官反射：当牵拉力量进一步增加，引起腱器官兴奋（阈值高于肌梭），抑制α运动神经元，抑制牵张反射
    • 放松被牵拉肌肉
    • 解除拮抗肌抑制

屈肌反射
...

  • 伤害性刺激作用于肢体皮肤时，受刺激侧肢体的屈肌收缩、伸肌舒张，肢体立即回缩
  • 意义：避开伤害性刺激

对侧伸肌反射
...

  • 如果受到的刺激较强，在同侧肢体屈曲的基础上（屈肌反射后0.2-0.5 s）可出现对侧肢体伸直的反射反应
  • 意义：保持身体平衡和维持躯体姿势

中枢模式发生器
...

  • 中枢模式发生器：脊髓中产生节律性运动活动（eg.控制协调的行走运动）的神经环路被称为
  • 产生机制：还没有一致的答案，最简单的模式发生器是单个具备起搏器特性的神经元

损伤表现
...

下运动神经元损伤
...

  • 所支配的骨骼肌
    • 软瘫：瘫痪、随意运动障碍、肌张力下降、腱反射减弱或消失
    • 肌萎缩、纤维颤动（Ach受体代偿）或肌束颤动
  • 常见于
    • 脊髓灰质炎：脊髓灰质炎病毒感染运动神经元
    • 肌萎缩侧索硬化症：俗称渐冻症，会有脊髓α-运动神经元的退行性变

脊休克
...

  • 脊休克：突然横断动物或人的脊髓（C5水平以下，保持呼吸），断面以下的脊髓暂时丧失反射活动的能力，而进入无反应状态
  • 主要表现：横断面以下躯体和内脏反射均减弱或消退
    • 脊髓支配的骨骼肌反射消失，肌张力下降或消失
    • 外周血管扩张，血压下降，发汗反射消失，粪尿潴留
    • 感觉减弱或消失
  • 恢复：脊休克是一种暂时现象，脊髓反射活动可逐渐恢复，但随意运动和感知觉永久丧失
    • 动物进化程度越高，恢复越慢
    • 简单和原始的反射（如屈肌反射）先恢复，复杂反射（如对侧伸肌反射）后恢复，血压逐渐恢复，并出现排便和排尿反射
    • 反射恢复后，有些比正常时增强（如屈肌反射），有些反射减弱（如对侧伸肌反射）
  • 机制：由于横断面以下的脊髓突然失去了高位中枢（如大脑皮层、脑干网状结构、前庭核）的下行调控作用
    • 证据：反射恢复后，再次切断脊髓不会再次引起脊休克，表明脊休克不是切断损伤刺激本身引起的
  • 表明：
    • 脊髓能完成某些简单反射
    • 正常时在高位中枢控制下活动（易化或抑制作用）
    • 正常脊髓整合来自外周和高位中枢两方面的信息，才能保证各种反射活动的正常进行和躯体运动的协调
